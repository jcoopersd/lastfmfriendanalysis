# Last.fm Music Discovery

Discover what your friends are listening to on Last.fm.

## Features
- Analyze your friends' listening habits across multiple time periods
- View top artists, albums, and tracks
- Sort by popularity, total scrobbles, or average position
- Filter by number of friends listening

## How to use
1. Get a free API key from [Last.fm](https://www.last.fm/api/account/create)
2. Enter your Last.fm username and API key
3. Wait for the data to load (takes a few minutes)
4. Explore what your friends are listening to!

## Privacy
Your API key is only used in your browser and is never stored or sent anywhere except to Last.fm's servers.
